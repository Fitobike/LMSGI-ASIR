HTML

HTML son las siglas HyperText Markup Language, que puede traducirse como "Lenguaje de Marcado de Hipertexto".

El lenguaje HTML se emplea para crear las páginas web y estructurar su contenido.Es muy facil ver el código HTML utilizando las herramientas de desarrollo del navegador.

HTML, se compone de etiquetas que se esacriben entre simbbolos de menor y mayor que.

Historia de HTML

Los inicios de HTML se remontan a 1990, cuando Tim Berners-Lee creó la primera página web.

Versiones de HTML


Tim Berners-Lee definió la primera versión de HTML, en el año 1991.

En la actualidad, la última versión de HTML es HTML5.
